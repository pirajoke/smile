## Name

## Short Description (132 chars max)
One-click AI-powered installation for any GitHub repo. Clone & set up projects instantly with Claude Code, Cursor, or Codex.

## Detailed Description
SMILE adds an "Install with AI" button to every GitHub, GitLab, and Bitbucket repository page.

One click to clone and set up any project using your favorite AI coding tool:
- Claude Code — AI-powered project setup
- Cursor — open repos directly in Cursor IDE
- Terminal + Claude — git clone with AI assistance
- Codex CLI — automated setup via OpenAI Codex

FEATURES:
- Smart stack detection (Node.js, Python, Rust, Go, Docker)
- Repository trust info (stars, license, last commit)
- NFT emoji lootbox — share repos with friends via Telegram, WhatsApp, or X
- Custom install commands
- Keyboard shortcut (Cmd+Shift+I / Ctrl+Shift+I)
- Dark mode support
- Zero tracking, zero analytics, fully open source

PRIVACY:
- No data collection
- No external requests
- No tracking or analytics
- Fully open source: github.com/pirajoke/smile
