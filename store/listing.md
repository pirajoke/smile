## Name
Ask your GIT — AI for any repo

## Short Description (132 chars max)
Ask AI about any GitHub repo — get instant summaries, chat about the code, and install with one click using Claude, Cursor, or Codex.

## Detailed Description
Ask your GIT adds a button to every GitHub, GitLab, and Bitbucket repository page.

Ask AI about any repo before you install it:
- Quick Summary — instant AI overview of what the project does
- Ask AI — interactive chat about the repo's code, features, and setup
- One-click install with Claude Code, Cursor, Codex CLI, or your own tools

FEATURES:
- AI-powered repo summaries (free, no API key needed)
- Interactive AI chat about any repository
- Smart stack detection (Node.js, Python, Rust, Go, Docker)
- Repository trust info (stars, license, last commit)
- 25+ tool presets (Windsurf, Zed, JetBrains, VS Code, and more)
- Custom install commands with placeholders
- Terminal Bridge — execute directly in your terminal
- Keyboard shortcut (Cmd+Shift+I / Ctrl+Shift+I)
- Dark mode support
- Zero tracking, zero analytics, fully open source

PRIVACY:
- No data collection
- No external tracking
- No analytics
- Fully open source: github.com/pirajoke/askyourgit
